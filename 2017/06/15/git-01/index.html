放弃修改

除了版本回退外，只要未commit之前都可以放弃工作区所做的修改。
如果有add 动作，需要先从暂存区回退到工作区git reset HEAD index.html
如果没有add动作，可直接放弃修改git checkout -- index.txt
或者可以这么理解git checkout -- index.txt 都会从暂存区或版本库(优先从暂存区)恢复到工作区.
